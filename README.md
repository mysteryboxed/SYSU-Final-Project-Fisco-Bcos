# 开发环境

* **OS** : MacOS
* **前端** : python+Qt5
* **IDE** : QtCreator


# 运行方式：
启动节点`bash nodes/127.0.0.1/start_all.sh`

将 **source** 文件夹内容放入配置好的python SDK 文件夹中，直接运行 **main.py**。部署方法参考[Python SDK](https://github.com/FISCO-BCOS/python-sdk)

运行 **main.py** 之后需要先进行账户的注册, 之后再进行转账等交易工作。具体可以参考演示视频中的内容
